# Customer Churn Prediction

Predicts customer churn using the Telco Customer Churn dataset (Kaggle).
Compares three ML models with full evaluation metrics and visualizations.

## Model Results
 Model               | Accuracy 
---------------------|----------
 Logistic Regression | ~ 0.738822   
 Random Forest       | ~ 0.788502    
 SVM                 | ~ 0.747339   

Best Model: Random Forest

## Evaluation Metrics
- Accuracy, Precision, Recall, F1 Score, ROC-AUC
- Confusion Matrix
- ROC Curve Comparison
- Feature Importance Analysis

## Visualizations
- Churn distribution
- Churn vs Contract Type
- Tenure distribution by Churn
- Monthly Charges vs Churn
- Correlation Heatmap

## Tech Stack
Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

## Dataset
[Telco Customer Churn – Kaggle](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)

## How to Run
Open in Google Colab and run all cells top to bottom.
